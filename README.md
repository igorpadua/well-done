# Bem-Feito

O bem-feito é uma lista de tarefas seguindo a ideia do À FAZER/FAZENDO/FEITO.

# Caracteristicas

* Simples e Objetivo;
* Não é possível remover uma tarefas;
* É possível filtrar por À FAZER/FAZENDO/FEITO
* É possível pesquisar pelo nome de um tarefa.
* Criado com Qt
